Mediaburst Nagios Plugins
-------------------------

A few plugins for the Nagios monitoring system

Licensed for use under the ISC license, a full copy can be found in
LICENSE.txt.

The Plugins
-----------

1. notifiy_sms.pl

Send SMS alerts when your monitors are critical.

2. check_smpp.pl

Monitor an SMPP server and make sure it is accepting messages.
You need the perl Net::SMPP module installed to use this plugin.
